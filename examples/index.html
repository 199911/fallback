Fallback JS Examples

Getting Started

An illustration of how to use the library in practice with jQuery and jQuery UI. The goal of this example is to simply get you started with a very basic example of how you can go about using the library with your project.

Loading Scripts

How to go about configuring JavaScript libraries to work with Fallback JS. This example also illustrates how you can go about setting up dependencies for your libraries, aliases and more.

Loading Stylesheets

How to go about configuring Stylesheets to work with Fallback JS. This example also shows you how to check to prevent a Stylesheet from loading if it's already been loaded on the page.

Loading Images

How to go about configuring Images to work with Fallback JS. This example shows you how to preload images with fallbacks, along with using inline Base64 images.

Angular

Illustration of how to use Angular JS with the Fallback JS library.

AMD Enforced

Illustration of how to enforce the use of AMD (Asynchronous Module Definitions) with the library.

Anonymous Modules

Illustration of how we can load files/modules without having any configuration set for them. Extremely useful when working on a project and spinning up new files that way we don't have to hand type out their configuration.

Error Handling

How to hanlde errors from the `define` and `require` functions.

Debugging

How to turn on debugging with the library.
